A)(i) Uploading and installing the theme - top

Open the Folder named "Theme". There are two folders that you need to upload in order to install the theme. One is in the SKIN folder and the other one in the APP folder. Both folders contain a folder named "dropshipper".

  1. Upload the folder "dropshipper" from the folder APP to your magento folder in app/ design/frontend/default/
  2. Upload the one in the SKIN folder to your magento folder in skin/frontend/default/

Go to admin » system » design. Click on "ADD DESIGN CHANGE" and select "dropshipper" for the custom design and click save.

A)(ii) Uploading the Module - top

Make sure Magento Cache is disabled.

Open the folder named "Module" from the "theme" folder.

  1. Upload the folder "Webizmu" to magento/app/code/local/
  2. Upload the file "Webizmu_Dropshipper.xml" to magento/app/etc/modules/

When you first install the module you will have to logout from admin and log back in. Go to system » configuration and you should see the module on top. See picture below:

A)(iii) Magento Error - top

If you refresh the homepage, Magento will throw an error if an attribute named "featured" has not been created. Follow the steps below to create the new attribute and add it to the default attribute set.

The steps consist of creating an attribute called Featured and adding it to the default attribute set.

  1. Go to admin » catalog » attributes » manage attributes
  2. Click Add new attribute
  3. Fill the text boxes like below:
    • Attribute code: featured
    • Scope : Store View
    • Catalog Input Type for Store Owner: YES/NO
    • Apply To: All product types
    • Use In Layered Navigation: No
    • Use In Search Results Layered Navigation: No
    • Visible on Product View Page on Front-end: No
    • Used in Product Listing: No
    • Used for Sorting in Product Listing: No
  4. Save the attribute and Go to catalog->attributes->manage attributes sets
  5. Click on Default and drag the featured attribute from the "Unassigned Attributes" under General. See picture below.
  6. Save Attribute set and add a new product. You should see a dropdown with YES/No. Selecting YES will show the product as featured on its category page.

Refresh the Homepage and the error is gone!

B) Theme Configuration - top

The theme's configuration are done from the admin panel in the "Dropshipper Module".

  1. Go to admin » system » configuration and you should see all the options.

B)(ii) Setup the Homepage - top

The homepage uses the 2columns-left layout.

  1. Go to admin » cms » pages.
  2. Click the design tab and change the page layout to "2 Columns with left bar".

B)(iii) Logo,styles & Favicon - top

Uploading your logo, changing the Theme color and uploading your Favicon is super easy.

Just go to admin » system » configuration and click the "Logo, Style & Favicon" tab and add your content.

Hit "Save Config" when you are done.

B)(iv) Uploading Banner Images - top

The banner area is configured from the Dropshipper Module in the admin section.

  1. Go to admin » system » configuration
  2. Click the "Homepage Slideshow" tab and upload your images.

To add alt text and image link for each uploaded image click "Homepage Slideshow Configuration". After that hit Save and refresh the homepage. You should see the images.

B)(v) Featured Products - top

To get "Featured" products to display on the Homepage and Category pages, all you have to do is to select "YES" next to the "Featured" label when editing or creating new products.

You can enable and disable the featured products from the "Dropshipper Module" section

To disable the Featured products just hit "Featured Product Options". When you disable you will get fields to adjust the homepage slideshow width and height.

B)(vi) Latest Products - top

The latest products will display automatically on the homepage. The products are sorted by latest first. No need to add "new to date". Just add your product information and hit save.

C) Languages - top

I have added the file "Mage_page.csv" for French language in the folder "French".

Download the french language pack from Magento Connect and replace the file "Mage_page.csv" with the one found in the folder "French".

For other languages you can grab them from Magento Connect, replace the "Mage_page.csv" with the one in the French folder and edit the french wordings to your language.
